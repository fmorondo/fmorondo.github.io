📝 Transcriptor de entrevistas

Sube un audio y descarga automáticamente las transcripciones en un zip cuando el sistema termina el procesamiento.

📘 Haz clic aquí para instrucciones, sobre todo si no lo has usado.

Este es un sistema en pruebas de transcripción de entrevistas y ruedas de prensa. En esta versión funciona de la siguiente forma:

  • Elige un archivo. Es mejor que le hayas puesto un nombre claro y que no llegue a media hora de duración.
  • Si necesitas cortarlo, puedes utilizar este servicioen la web: https://mp3cut.net/es/
  • Selecciona el idioma original y la plantilla (prompt) que se usará para corregir/formatear. Hay versiones para entrevistas y ruedas de prensa o conferencias. La versión para ruedas de prensa está todavía en una fase preliminar.
  • Pulsa en Subir el archivo y procesarlo.
  • El sistema va informando de los pasos que da.
  • Al final se descargará el ZIP automáticamente.
